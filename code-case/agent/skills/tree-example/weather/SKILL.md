# 天气路由 Skill

## 子 Skill

- `travel`：当用户的问题和旅游、旅行、景点、酒店、航班、海边或山区出行相关时使用。
- `commute`：当用户的问题和上班、上学、开车、步行、公交、地铁通勤相关时使用。
- `outdoor-sport`：当用户的问题和跑步、骑行、徒步、登山、露营等户外运动相关时使用。

## 路由规则

1. 根据用户意图选择一个最合适的子 Skill。
2. 如果用户意图已经明确，可以直接进入更深层的叶子 Skill。
3. 如果无法判断旅游、通勤还是户外运动，只问一个澄清问题。

## 示例

用户：

```text
周末去青岛海边玩，天气适合吗？
```

路由结果：

```text
weather -> travel
```

用户：

```text
明天开车上班会不会受天气影响？
```

路由结果：

```text
weather -> commute
```
