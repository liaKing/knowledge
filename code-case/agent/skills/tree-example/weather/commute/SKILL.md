# 通勤天气路由 Skill

## 子 Skill

- `driving`：当用户提到开车、驾车、路况、能见度、路面湿滑或早晚高峰时使用。
- `walking`：当用户提到步行、骑共享单车、是否需要雨伞、穿衣时使用。
- `public-transit`：当用户提到公交、地铁、换乘、等车或延误时使用。

## 路由规则

1. 选择和用户通勤方式最匹配的子 Skill。
2. 如果用户没有说明通勤方式，先询问是开车、步行还是公共交通。
3. 当前示例只实现 `driving` 叶子 Skill。

## 示例

用户：

```text
明天开车上班会不会受天气影响？
```

路由结果：

```text
commute-weather -> driving
```
