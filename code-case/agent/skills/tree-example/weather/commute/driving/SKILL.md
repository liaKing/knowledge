# 开车通勤天气 Skill

## 执行流程

1. 提取通勤城市或路线。
2. 如果地点缺失，询问用户所在城市或通勤路线。
3. 运行脚本获取固定通勤天气结果。
4. 重点关注降雨、能见度、风力和路面风险。
5. 给出简短的驾驶建议。

## 脚本

执行命令：

```bash
go run scripts/fixed_driving_weather.go --city "<city>"
```

## 输出格式

回答包含：

- 天气概况
- 驾车风险
- 通勤建议

## 示例

```text
北京明早多云，能见度正常，路面风险较低。可以正常开车通勤，建议预留 10 分钟缓冲时间。
```
