# 旅游天气路由 Skill

## 子 Skill

- `beach`：当用户提到海边、海岛、沙滩、游泳、晒太阳、海风、紫外线时使用。
- `mountain`：当用户提到登山、徒步、山区、海拔、露营、山路时使用。
- `city-trip`：当用户提到城市游、景点打卡、博物馆、餐厅、购物、城市步行时使用。

## 路由规则

1. 选择最具体的子 Skill。
2. 如果用户提到“海边”“海岛”“沙滩”，优先进入 `beach`。
3. 如果用户提到“山”“徒步”“露营”，优先进入 `mountain`。
4. 如果只是普通旅游天气，可以按通用旅游天气处理，或询问用户具体旅行类型。

## 示例

用户：

```text
周末去青岛海边玩，天气适合吗？
```

路由结果：

```text
travel-weather -> beach
```
