# 海边旅游天气 Skill

## 执行流程

1. 提取用户要去的城市、海边、海岛或景点名称。
2. 如果地点缺失，询问用户具体地点。
3. 运行脚本获取固定海边天气结果。
4. 重点关注风力、紫外线、降雨、温度和是否适合下水。
5. 给出简短的海边出行建议。

## 脚本

执行命令：

```bash
go run scripts/fixed_beach_weather.go --place "<place>"
```

## 输出格式

回答包含三部分：

- 天气概况
- 海边风险提示
- 一句出行建议

## 示例

```text
青岛海边今天晴到多云，约 26°C，海风较明显。适合海边游玩，但不建议长时间暴晒，是否下水要看当天海浪和景区提示。
```
